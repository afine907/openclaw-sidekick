## Freqtrade 加密货币交易机器人安装

### 安装过程
1. 系统环境：Alibaba Cloud Linux，Python 3.6（太低）
2. 安装 Python 3.11：`sudo dnf install -y python3.11 python3.11-pip python3.11-devel`
3. 克隆并安装 freqtrade：
   ```bash
   cd /home/admin && git clone https://github.com/freqtrade/freqtrade.git
   cd freqtrade && bash setup.sh -i
   ```
4. 安装路径：`/home/admin/freqtrade`

### 配置
- 配置文件：`/home/admin/freqtrade/user_data/config.json`
- 交易所：Gate.io（Binance 因地理限制无法访问，Kraken 不支持 klines 直接下载）
- API 端口：8081（避免与 SearXNG 的 8080 冲突）
- 认证：freqtrade / freqtrade
- 策略：SampleStrategy（默认 RSI 策略）
- 运行模式：Dry-run（模拟交易，初始资金 1000 USDT）

### 运行命令
```bash
cd /home/admin/freqtrade && source .venv/bin/activate
# 下载历史数据
freqtrade download-data -c user_data/config.json --pairs BTC/USDT ETH/USDT --timeframes 5m --days 30
# 回测
freqtrade backtesting -c user_data/config.json --strategy SampleStrategy
# 启动 dry-run 交易
freqtrade trade -c user_data/config.json --strategy SampleStrategy --dry-run
```

### 回测结果（30天，BTC/USDT + ETH/USDT）
- 总交易次数：29
- 胜率：48.3%
- 总收益：-5.85%（亏损）
- 默认策略需要优化

### API 端点
- Ping: `curl http://localhost:8081/api/v1/ping`
- 状态: `curl -u freqtrade:freqtrade http://localhost:8081/api/v1/status`
- 配置: `curl -u freqtrade:freqtrade http://localhost:8081/api/v1/show_config`

- 策略优化和参数调优
