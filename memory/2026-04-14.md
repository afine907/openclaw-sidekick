## 全局安装 Claude Code

```bash
sudo npm install -g @anthropic-ai/claude-code
```

版本：2.1.105

### 百度千帆 Coding Plan 配置

配置文件：`~/.claude/settings.json`

```json
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "bce-v3/ALTAKSP-xxx",
    "ANTHROPIC_BASE_URL": "https://qianfan.baidubce.com/v2/coding",
    "ANTHROPIC_MODEL": "qianfan-code-latest",
    "API_TIMEOUT_MS": "600000",
    "CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC": "1"
  }
}
```

使用：运行 `claude` 命令即可直接使用千帆模型

### 问题与解决

**问题**：百度千帆 Coding Plan 只支持 OpenAI 格式的 `/chat/completions` 端点，不支持 Anthropic 格式的 `/messages` 端点。

**解决方案**：创建了自定义代理服务 `openclaw-claude-code-proxy.py`

代理位置：`/home/admin/.openclaw/workspace/openclaw-claude-code-proxy.py`

Systemd 服务：`openclaw-claude-code-proxy.service`（已启用并运行）

管理命令：
```bash
sudo systemctl status openclaw-claude-code-proxy
sudo systemctl restart openclaw-claude-code-proxy
sudo journalctl -u openclaw-claude-code-proxy -f
```
