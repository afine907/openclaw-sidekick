# 任务分工指南：本地 vs 服务器

> 服务器：计算、存储、API 调用 | 本地：浏览器、图形界面、爬虫

---

## 服务器擅长 ✅

| 任务类型 | 说明 | 示例 |
|---------|------|------|
| **代码编译** | CPU 密集型，服务器性能强 | 编译大型项目、构建 Docker 镜像 |
| **数据处理** | 内存大，适合批处理 | 分析日志、处理 CSV/JSON |
| **API 调用** | 稳定网络，无 IP 限制 | OpenAI API、GitHub API（认证后） |
| **文件存储** | 大容量磁盘 | 代码仓库、文档、备份 |
| **定时任务** | 24小时运行 | Cron 定时备份、监控 |
| **AI 模型推理** | GPU 或 CPU 推理 | 运行本地模型 |
| **数据库** | 持久化存储 | PostgreSQL、Redis |

---

## 本地电脑擅长 ✅

| 任务类型 | 说明 | 原因 |
|---------|------|------|
| **浏览器自动化** | 需要 GUI | 服务器无显示器 |
| **视频下载** | YouTube/B站 | 云 IP 被封锁 |
| **社交媒体登录** | 扫码/手机验证 | 需要图形界面 |
| **OCR 识别** | 截图识别 | 需要显示器 |
| **视频剪辑** | 视频处理 | 需要 GPU + GUI |
| **游戏相关** | 游戏自动化 | 需要图形渲染 |

## 具体场景对照表

### 爬虫/数据抓取

| 目标 | 适合位置 | 原因 |
|------|---------|------|
| YouTube 字幕 | 本地 | 云 IP 被封锁 |
| B站视频/字幕 | 本地 | 同上 |
| 小红书登录 | 本地 | 需要扫码 |
| 微信/飞书 API | 服务器 | 官方 API，无限制 |
| GitHub API | 服务器 | 配 Token 后正常 |
| 网页爬虫 | 看情况 | 有反爬则本地 + VPN |

### 文档处理

| 任务 | 适合位置 | 工具 |
|------|---------|------|
| PDF 转文字 | 服务器 | nano-pdf |
| Word/Excel 生成 | 服务器 | word-docx / excel-xlsx |
| OCR 图片识别 | 本地优先 | 需要截图场景 |
| 视频帧提取 | 服务器 | ffmpeg |

### 开发相关

| 任务 | 适合位置 | 说明 |
|------|---------|------|
| 代码编写/调试 | 本地 | IDE 体验好 |
| 代码编译/构建 | 服务器 | 性能强 |
| Git 操作 | 服务器 | 推送到 GitHub |
| CI/CD 运行 | 服务器 | 自动化流程 |
| 测试运行 | 服务器 | 批量测试 |

## 混合方案

**本地处理 → 上传服务器 → 服务器处理**

```
┌─────────────────┐      ┌─────────────────┐
│   本地电脑       │      │   云服务器       │
│                 │      │                 │
│ 1. 下载字幕     │ ───► │ 2. 分析/总结    │
│ 3. 小红书登录   │ ───► │ 4. MCP 服务运行 │
│ 5. 扫码登录     │ ───► │ 6. API 调用     │
└─────────────────┘      └─────────────────┘
        │                        │
        │      上传文件/cookies   │
        │ ◄─────────────────────►│
        └────────────────────────┘
```

**传输命令：**
```bash
# 本地 → 服务器
scp cookies.json admin@服务器IP:~/xiaohongshu-mcp/
scp video.vtt admin@服务器IP:/tmp/

# 服务器 → 本地
scp admin@服务器IP:/tmp/result.json ./
```

## 当前环境的限制

| 限制 | 影响 | 解决方案 |
|------|------|---------|
| 无显示器 | 无法运行 GUI 程序 | 本地运行 → 上传结果 |
| 云 IP 被封锁 | YouTube/B站无法访问 | 本地下载 + VPN |
| Python 3.6 系统默认 | 部分工具不兼容 | 使用 Python 3.11 |
| 无代理 | 无法绕过 IP 封锁 | 配置代理或本地处理 |

---

## 快速决策流程图

```
任务是否需要浏览器/GUI？
├─ 是 → 本地处理
│        └─ 结果上传服务器
│
└─ 否 → 是否需要绕过 IP 封锁？
         ├─ 是 → 本地 + VPN 处理
         │
         └─ 否 → 服务器处理 ✓
```
