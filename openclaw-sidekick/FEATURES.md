## 三、技术方案

### 3.1 架构升级

```
src/
├── components/          # UI 组件
│   ├── onboarding.js    # 新手引导
│   ├── commandPalette.js # 命令面板
│   ├── codeBlock.js     # 代码块
│   └── statusBar.js     # 状态栏
├── features/            # 功能模块
│   ├── commands.js      # 快捷指令
│   ├── presets.js       # 预设模板
│   ├── history.js       # 对话历史
│   └── plugins.js       # 插件系统
├── utils/               # 工具函数
│   ├── logger.js        # 日志
│   ├── diagnose.js      # 诊断
│   └── exporter.js      # 导出
└── ...
```

### 3.2 存储结构升级

```javascript
{
  "settings": {
    "apiEndpoint": "http://localhost:4000",
    "theme": "dark",
    "shortcuts": {...}
  },
  "presets": [
    {"id": "1", "name": "解释代码", "prompt": "请解释这段代码..."}
  ],
  "history": [...],
  "plugins": [...]
}
```
